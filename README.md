如果您需要找到小程序源码，可以打开[blog-weapp](https://github.com/cumt-robin/blog-weapp)。

我写了一系列博客专栏，介绍如何使用 Vue3+TS+Node 打造个人博客。[点击前往专栏](https://juejin.cn/column/7177402980180688952)。

## 启动项目

本项目适合 Node 16 版本。

安装依赖：

```
yarn
```

启动项目：

```
yarn serve
```

## 打包项目

```
yarn build
```
